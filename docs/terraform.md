# Command Line Deployment via Terraform

For those who prefer deploying via the command line, we provide a Terraform module that simplifies the process. This module allows you to programmatically provision the SQL Server 2022 Developer Edition virtual machine on Google Cloud Platform.

## Using the Terraform Module

To get started, refer to the detailed instructions in the `README.md` file located within the Terraform module directory. It contains all the necessary information on prerequisites, input variables, and usage examples.

[View the Terraform Module README on GitHub](https://github.com/gclouds-co-uk/SQL-Server-2022-Developer/tree/main/example/terraform)
